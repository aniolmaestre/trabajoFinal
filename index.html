Descripción del proyecto

Esta página web trata sobre el trabajo final grupal de desarrollo web.
Nosotros hemos seleccionado la base de datos de Rick y Morty, que es una serie de televisión de animación 2D para adultos, donde se siguen las desventuras de un científico y su nieto con humor negro y ciencia-ficción. Tiene unos colores vivos, saturados y locos como en nuestra web.

El proyecto trata sobre la implementación de la API de Rick y Morty. Hemos escogido esta API debido a que nos parece completa y con una buena base con información para poder hacer distintas búsquedas.

Programario

En inglés "HyperText Markup Language", es un lenguaje de marcado para la creación de páginas web. Se define con una estructura básica y código HTML para la presentación de contenido de una página web.

Los elementos creados con HTML se pueden configurar visualmente con CSS. El formato es en cascada, donde se escoge el elemento y con unas reglas se modifica en vertical.

Es el lenguaje de programación interpretado que hemos usado para el proyecto. Su uso principal es en aplicaciones web y permite mejoras de interfaz de usuario y páginas web dinámicas.
